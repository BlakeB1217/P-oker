# P-oker

Heads-up poker strategy trainer with a rule-based coaching engine and ML-powered tendency analysis.

## Requirements

- [Node.js](https://nodejs.org) (v18+)
- [Python 3.9+](https://python.org)

## Run

```bash
chmod +x start.sh
./start.sh
```

Then open http://localhost:5173.

That's it — the script installs all dependencies and starts both the frontend and backend automatically.
